# Справа №13 — гра-привітання для Паули

Детективна top-down гра на Phaser 3: зібери 6 улик у маєтку, знайди 3 приховані спогади від Віки та Андрія, відкрий скриню в підвалі — і отримай привітання з Днем народження.

## Структура проєкту

```
paula-birthday-game/
├── index.html              ← вся гра (HTML + CSS + JS в одному файлі)
├── assets/
│   └── portraits/
│       ├── paula.png       ← фото Паули
│       ├── vika.png        ← фото Віки
│       └── andrii.png      ← фото Андрія
└── README.md
```

## Як додати або замінити фото

1. Поклади квадратні/портретні PNG у `assets/portraits/`.
2. Імена файлів мають бути **точно**: `paula.png`, `vika.png`, `andrii.png` (маленькі літери).
3. Найкраще — вирізані фігурки на прозорому фоні.

## Локальний перегляд

**Варіант A — термінал (рекомендовано):**

```powershell
cd paula-birthday-game
npx --yes serve -l 3000
```

Відкрий у браузері: `http://localhost:3000`

**Варіант B — VS Code Live Server:**

1. Встанови розширення **Live Server** (Ritwick Dey).
2. Правий клік на `index.html` → **Open with Live Server**.

> Не відкривай `index.html` подвійним кліком — Phaser завантажує ресурси через HTTP, і без сервера зображення можуть не підвантажитись.

## Керування

| Дія | Клавіші |
|-----|---------|
| Рух | WASD або стрілки |
| Взаємодія з об'єктами | `E` або клік |
| Закрити діалог | `E` або `Enter` |

## Що можна легко змінити в `index.html`

- **Текст привітання** — блок `id="finale-post"`.
- **Улики (6 шт.)** — масив `clueData` у скрипті.
- **Приховані спогади (3 шт.)** — масив `eggData`.
- **Назви кімнат** — масив `roomDefs`.
- **Колір/шрифт** — блок `:root { ... }` на початку файлу.

## Деплой на Vercel

### Варіант A: через сайт Vercel

1. Зайди на **vercel.com** → Sign Up → увійди через GitHub.
2. Завантаж проєкт на GitHub і підключи репозиторій у Vercel.
3. Framework Preset — **Other** (статичний сайт).
4. Натисни **Deploy**. Отримаєш посилання типу `https://paula-birthday-game.vercel.app`.

### Варіант B: через Vercel CLI

```powershell
npm install -g vercel
vercel login
vercel
vercel --prod
```
